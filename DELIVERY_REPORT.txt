================================================================================
                              一、项目概述
================================================================================

本项目是一个企业级的腾讯文档在线表格与MySQL数据库双向同步解决方案。
通过实时Webhook和定时轮询两种模式，实现腾讯文档与MySQL数据库的自动同步。

================================================================================
                          五、API接口清单
================================================================================

配置管理:
  GET    /api/configs                    - 获取配置列表
  POST   /api/configs                    - 创建配置
  GET    /api/configs/{id}               - 获取配置详情
  PUT    /api/configs/{id}               - 更新配置
  DELETE /api/configs/{id}               - 删除配置
  POST   /api/configs/{id}/test          - 测试连接

同步操作:
  POST   /api/sync/{id}/trigger          - 触发同步
  POST   /api/sync/{id}/to-mysql        - 单向同步到MySQL
  POST   /api/sync/{id}/from-mysql      - 单向同步到腾讯文档
  GET    /api/sync/{id}/status           - 获取同步状态

MySQL浏览器:
  GET    /api/mysql/databases             - 获取数据库列表
  GET    /api/mysql/databases/{db}/tables - 获取表列表
  GET    /api/mysql/tables/{table}/columns - 获取表结构

系统:
  GET    /health                         - 健康检查
  POST   /init                           - 初始化系统
  GET    /docs                           - API文档

================================================================================
                          八、部署指南
================================================================================

环境要求:
  - Python 3.10+
  - MySQL 8.0+
  - 4GB+ RAM
  - 10GB+ 磁盘空间

快速部署:
  1. 安装依赖:
     pip install -r requirements.txt

  2. 配置数据库:
     mysql -u root -p < migrations/init.sql

  3. 配置应用:
     编辑 config.yaml
     编辑 .env (API密钥等)

  4. 启动服务:
     python -m uvicorn app.main:app --host 0.0.0.0 --port 8083

  5. 访问:
     管理界面: http://localhost:8083/
     API文档: http://localhost:8083/docs
